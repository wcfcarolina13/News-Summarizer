User Guide

⚠ Alpha Version — This app is in early testing. API keys and Google credentials are currently managed by the admin. All API costs are on the admin's billing.

Getting Started

  1. Scheduler — Set up automated feeds that run on a schedule and export to Google Sheets
  2. News — Fetch and summarize YouTube videos with AI
  3. Extract — Pull article links from newsletters (ExecSum, CryptoSum, RWA, etc.)
  4. Audio — Generate audio briefings from your summaries
  5. Settings — Manage API key, sources, and custom instructions

Scheduler — Automated Feeds

The Scheduler runs tasks automatically (hourly, daily, weekly) and exports results to Google Sheets.

  • Add Task — Give it a name, source URL, and extraction config
  • Source URL — A Telegram channel (t.me/channel), newsletter archive, or RSS feed
  • Config — Choose a config to control what gets extracted and how columns are structured
  • Google Sheets — Toggle on, paste your spreadsheet URL/ID, and set the sheet tab name
  • Custom Columns — Override the default columns from the config. Uncheck "Use config defaults" and enter one column name per line
  • Run Now — Trigger an immediate extraction without waiting for the schedule

Data Extraction

Extract article links from newsletters and news feeds. Supports:

  • Beehiiv newsletters (ExecSum, CryptoSum)
  • Substack, Medium, news sites
  • Telegram channels (RWA, crypto news)
  • Grid integration for entity matching
  • CSV and Google Sheets export

  • YouTube channels (any public channel)

Troubleshooting

  • No summaries? — Check that sources are configured in Settings
  • API errors? — Verify your API key is correct
  • Sheets not working? — Ensure the spreadsheet is shared with the service account email
  • Task vanished? — Tasks persist to Google Sheets and survive redeployments. If tasks are missing, check SCHEDULER_SHEET_ID env var is set.
  • Extraction empty? — Some sites block automated access. Try a different URL or config.

Desktop App

The desktop app provides advanced features not available in the web dashboard:

  • Kokoro TTS — High-quality neural text-to-speech with multiple voices
  • Whisper Transcription — Convert audio/video files to text
  • Local audio playback — Preview and manage generated audio
  • Offline mode — Works without internet connection
  • Cloud sync — Connects to this web scheduler for 24/7 automation

Setup

  1. Download the app from GitHub Releases
  2. macOS: Open the .dmg and drag to Applications
  3. Windows: Run the .exe installer
  4. Enter your Gemini API key in Settings
  5. To connect to this cloud scheduler: go to Scheduler → Cloud Scheduler → enter this server URL
